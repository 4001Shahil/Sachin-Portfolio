## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

### Deploy to Vercel

1. Push your code to GitHub
2. Import the repository in Vercel
3. Deploy with one click

Or use Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Customization

- Update personal information in `src/constants/index.js`
- Add your resume PDF to `public/` folder
- Replace profile image in `public/` folder
- Configure EmailJS in `src/components/Contact.jsx`
